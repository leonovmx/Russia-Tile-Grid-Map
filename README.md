# Плиточная карта России (Russia Tile Grid Map)

Цель сделать для себя карту для отрисовки разных данных. Пока просто 4 типа и 4 цвета, для примера просто генерирую рандомные данные. 

Данные для отрисковки были взяты вне, ничего не убавлял и не добавлял. Как отмечено в [5] "у автора из РФ нет выбора, от слова совсем".
